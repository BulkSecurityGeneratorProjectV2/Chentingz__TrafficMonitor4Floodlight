# TrafficMonitor4Floodlight
添加了流量监控模块的Floodlight控制器

## 已完成：
* 2018/4/30 添加用于查询交换机端口统计信息的REST API,输入`http://localhost:8080/wm/trafficmonitor/portstats/<dpid>/<portno>/json`即可获取指定交换机端口的统计信息
